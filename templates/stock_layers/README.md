# Stock Layers

This directory defines the current layered stock model.

## Files

- `strict_buyable_stock_inchikeys.txt`: Small, conservative stock layer.
- `trusted_intermediate_stock_inchikeys.txt`: Literature- or family-supported intermediates such as aglycones/chalcones.
- `virtual_bridge_stock_inchikeys.txt`: Diagnostic bridge layer. Entries here validate route connectivity but do not prove buyability.
- `stock_layers_metadata.csv`: Evidence, role, and provenance metadata for all layered stock entries.
- `sugar_gap_clusters.csv`: Route-gap sugar classification table generated by `scripts/build_sugar_bridge_layer.py`.
- `sugar_bridge_stock.csv`: Route-gap-derived non-aromatic sugar bridge entries.

## Interpretation

`virtual_bridge` is intentionally not strict stock. A route using this layer should be reported as `bridge-closed`, not as real strict solved.

The current sugar bridge layer only auto-bridges non-aromatic sugar-like leaves. Aromatic flavonoid glycoside leaves are manual review targets and must not be promoted automatically.
